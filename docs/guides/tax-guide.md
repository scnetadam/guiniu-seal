|--------|---------|---------|------|----------|
| **A轨** | 工资薪金（与平台/公司存在雇佣关系） | 雇主代扣代缴 | ❌ 不需要 | — |

## 2. 触发规则

### B轨 — 劳务报酬

```
单笔金额 ≤ 800 元
  └─ 暂不扣缴，月底汇总（bucket）
   
单笔金额 > 800 元
  └─ 即时预扣 20%

本月累计 > 10,000 元
  └─ 强制预扣（应税部分 = 金额 - 800）× 20%
  └─ 风险标签: "月累超限: 建议引导C轨"
   
单日 ≥ 5 笔
  └─ 风险标签: "高频交易警戒"
```

## 3. 结算示例

### 案例 A：B轨小额（月内首笔）

```
交易：1,500 元（新能源车评测）
月累计：0 元
计算：
  - 单笔 > 800 → 即时预扣 20% = 300 元
  - 实际到账：1,200 元
  - 需要发票：✅
结果：实发 1,200 元，预扣 300 元（月底报税）
```

### 案例 B：B轨月累超限

```
交易：8,000 元（深度评测）
月累计：3,000 元（已发生）
计算：
  - 累计 = 11,000 > 10,000 → 触发强制预扣
  - 应税部分 = 8,000 - 800 = 7,200 元
  - 预扣 = 7,200 × 20% = 1,440 元
  - 实际到账：6,560 元
  - 风险标签: "月累超限"
结果：实发 6,560 元，预扣 1,440 元
```

### 案例 C：C轨经营所得

```
交易：15,000 元（季度深度评测合作）

  - Hash 证据链存证（防重新定性）
```

## 4. 优化建议

### 对于月入 > 1万的 KOL
- **推荐 C轨**：注册个体工商户，自行开票
- **优势**：全额拨付，无预扣，税务合规
- **平台支持**：自动生成 Hash 证据链，防止被重新定性为劳务报酬

### 对于月入 < 1万的 KOC
- **B轨小额**：利用 800 元以下免预扣额度
- **月底汇总**：多笔小额在月底一次性处理

### 对于企业内容团队
- **A轨**：与平台签订雇佣/劳务协议，雇主代扣

## 5. 合规提醒

⚠️ **不得通过分拆交易规避预扣**
- 单笔 ≤ 800 但同一日多笔 → 触发高频警戒
- 月累 > 1万但通过分拆规避 → 税务风险

⚠️ **C轨需要真实经营实质**
- 必须有个体工商户营业执照
- 必须实际开具发票
- 平台保留 Hash 证据链，供税务核查
